# Rental Posting Website

A comprehensive platform that allows users to post rental listings and search for suitable rooms with ease. 

**Timeline:** 01/2025 - 03/2025

## 🚀 Technologies Used

* **Frontend:** ReactJS, Tailwind CSS
* **Backend:** Node.js (ExpressJS)
* **API:** RESTful API connecting frontend and backend

## ✨ Main Features

### 👤 User Features
* **Authentication:** Secure user registration and login functionality.
* **Rental Listings Management:** Create and post rental listings including title, images, price, area, location, and detailed description.
* **Advanced Search & Filter:** Search for rooms and filter results by area, price range, and amenities.
* **Post Management:** Manage the reliability of personal posts (edit, delete listings).

### 🛡️ Admin Features
* **User Management:** Oversee and manage users and accounts across the platform.
* **Category Management:** Manage and organize rental categories.
